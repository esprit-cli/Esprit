## Setup

```bash
export ESPRIT_LLM="anthropic/claude-opus-4-7"
export LLM_API_KEY="sk-ant-..."
```

## Available Models

| Model | Description |
|-------|-------------|
| `anthropic/claude-opus-4-7` | Most capable model for complex reasoning and agentic coding (recommended) |
| `anthropic/claude-sonnet-4-6` | Best balance of speed and intelligence |
| `anthropic/claude-haiku-4-5-20251001` | Fastest Claude option |

## Get API Key

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Navigate to API Keys
3. Create a new key
