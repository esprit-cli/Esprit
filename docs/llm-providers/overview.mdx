Esprit uses [LiteLLM](https://docs.litellm.ai/docs/providers) for model compatibility, supporting 100+ LLM providers.

## Recommended Models

For best results, use one of these models:

| Model             | Provider      | Configuration                    |
| ----------------- | ------------- | -------------------------------- |
| GPT-5            | OpenAI        | `openai/gpt-5`                   |
| Claude Opus 4.7  | Anthropic     | `anthropic/claude-opus-4-7`      |
| Gemini 3 Pro     | Google Vertex | `vertex_ai/gemini-3-pro-preview` |
| MiniMax M2.5 Free| OpenCode Zen  | `opencode/minimax-m2.5-free`     |

## Quick Setup

```bash
export ESPRIT_LLM="openai/gpt-5"
export LLM_API_KEY="your-api-key"
```

For OpenCode public free models, credentials may not be required when public models are detectable locally:

```bash
export ESPRIT_LLM="opencode/minimax-m2.5-free"
```

If a public OpenCode model hits rate limits or upstream instability, Esprit automatically falls back to another public model.

## Model Format

Use LiteLLM's `provider/model-name` format:

```
openai/gpt-5
anthropic/claude-opus-4-7
vertex_ai/gemini-3-pro-preview
bedrock/anthropic.claude-4-5-sonnet-20251022-v1:0
ollama/llama4
```
